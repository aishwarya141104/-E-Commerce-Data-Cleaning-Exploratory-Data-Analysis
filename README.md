# 🌟 Project Overview

In today's data-driven business environment, raw data is rarely analysis-ready. This project demonstrates a complete **Data Cleaning** and **Exploratory Data Analysis (EDA)** workflow using Python to transform raw E-Commerce sales data into meaningful business insights.

The project covers every stage of the analytics pipeline, including data preprocessing, handling missing values, duplicate detection, exploratory analysis, visualization, and exporting a clean dataset for reporting and dashboard development.

---

# 🎯 Objectives

✔ Import and inspect raw dataset

✔ Clean inconsistent data

✔ Handle missing values

✔ Remove duplicate records

✔ Detect potential outliers

✔ Perform Exploratory Data Analysis

✔ Visualize business metrics

✔ Generate actionable business insights

✔ Export cleaned dataset

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/aishwarya141104/E-Commerce-Data-Cleaning-EDA.git
```

Move into the project directory

```bash
cd E-Commerce-Data-Cleaning-EDA
```

Install dependencies

```bash
pip install pandas numpy matplotlib openpyxl
```

Launch Jupyter Notebook

```bash
jupyter notebook
```

Open

```
Data_Cleaning_Project.ipynb
```

Run all notebook cells.
